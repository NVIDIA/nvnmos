# Other Ways to Use NvNmos

Use the [C API](https://nvidia.github.io/nvnmos/) to embed the NMOS control
plane directly in a Media Node application.

Use the
[daemon guide and gRPC API reference](https://nvidia.github.io/nvnmos/grpc/)
to run the control plane out of process and connect Media Node applications to
`nvnmosd` over gRPC.
